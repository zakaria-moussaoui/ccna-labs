## Objective
1. Block network 192.168.10.0/24 to reach 192.168.30.0/24 (Standard ACL)
2. Block ping 192.168.20.10 to reach 192.168.50.10 (Extended ACL)

NB: This lab covers the configuration of **Access Control Lists (ACL)** on Cisco routers.
ACLs are used to filter network traffic by permitting or denying packets
based on defined rules.

Two types of ACL are practiced in this lab:

- **Standard ACL** → filters traffic based on source IP address only
- **Extended ACL** → filters traffic based on source/destination IP, protocol, and port

Routing protocol used : **OSPF**
Simulator used : **Cisco Packet Tracer**
